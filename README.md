## Run Locally

Clone the project

```bash
  git clone https://github.com/Sagyam/Poll-App.git
```

Go to the project directory

```bash
  cd Poll-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
